How do I download the source?

In order to download the source code for all of our projects go to https://github.com/StructML and extract the zip from one of our repositories. If you lack a GUI type:

"git clone https://github.com/StructML/REPOSITORY-NAME"


If you don't have Git, install it by typing on Debian based systems:

"sudo apt-get install git"
or on RPM systems

"yum install git-all"
